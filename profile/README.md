# Radarboard

Radarboard is a desktop board for tracking the signals that matter across code,
ops, growth, and community work.

## Repositories

- [`radarboard`](https://github.com/Radarboard/radarboard) is the core monorepo.
- [`homebrew-radarboard`](https://github.com/Radarboard/homebrew-radarboard) is
  the Homebrew tap for desktop releases.
- [`community-extensions`](https://github.com/Radarboard/community-extensions)
  is the future home for community-maintained integrations, plugins, and widgets.
